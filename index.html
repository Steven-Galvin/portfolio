While I've only recently begun my schooling, I've been fond of web design since my freshamn year of High School back in 2009. I hope to further advance and refine my skills in this field so I can get out of Costco and into doing something I am passionate about.

Other Works

Here is a short list of my projects from newest to oldest. (All links are to my Github repositories for you to access and clone the code.)

  1. Wikipedia Recreate

    This is my latest project. A showcase of all I have learned so far at Epicodus. My partner, Emily Dokken, and I were tasked with recreating the Wikipedia homepage using what we knew of HTML, CSS, and Bootstrap. While not a perfect recreation, it is a good example of our understanding on how to create and style a basic website.

  2. Vacation

    This was a project made to practice my skills with Bootstrap. It is unfinished as of now, but it shows that I understand the basics of styling with Bootstrap by using containers, rows, wells, and more to create webpage detailing my dream vacation.

  3. Animal Shelter

    To display our knowledge of the box model, this project was made by Arianna Chau and I. It is a fictional animal shelter page housing four cute animals that need rescuing. We used floats and the box model to create this simple and easy to view webpage.

  4. Dogblog

    One of my earlier projects. The purpose of this project was to practice basic HTML tags with the introduction of images. It's clearly a beginners page, but it's a good example of how far I've come within four short weeks.
